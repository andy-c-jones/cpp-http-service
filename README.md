# Requirements

Installing dependencies
Ubuntu
```sudo apt-get install build-essential libtcmalloc-minimal4 && sudo ln -s /usr/lib/libtcmalloc_minimal.so.4 /usr/lib/libtcmalloc_minimal.so```

OSX
```brew install boost google-perftools```
